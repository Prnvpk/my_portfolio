Project details

Pothole Detection System

A computer vision workflow with a mobile interface for real-time road hazard detection and alerts.

Python YOLOv7 Flutter Roboflow
Project details

Designed and developed a mobile application for real-time pothole detection by integrating the YOLOv7 deep learning model with a Flutter-based frontend. Created and annotated a custom dataset using Roboflow to improve detection accuracy, and trained the model using Python for precise identification of potholes with labeled outputs. Implemented a real-time alert system to notify drivers about detected road hazards, enhancing safety and awareness. The application features an intuitive interface built with Flutter, ensuring smooth interaction between the user and the detection system.
